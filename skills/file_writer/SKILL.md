# 文件写入技能

## 何时使用

当用户希望你在当前工作区里创建、重写或更新本地文件时，使用这个技能。

## 输入

- 目标文件路径
- 期望写入的内容
- 格式、结构或风格约束

## 步骤

1. 如果周边目录结构会影响结果，先用 `list_dir` 查看上下文。
2. 如果是改已有文件，先用 `read_file` 读取当前内容。
3. 用 `write_file` 写入新内容。
4. 如有必要，再次读取文件确认写入结果。

## 工具偏好

优先使用 `read_file`、`write_file`、`list_dir`。只有任务明确需要 Shell 能力时，才使用 `run_command`。
